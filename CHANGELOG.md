## [0.1.0] - 2025-01-03

### Initial Release

#### Features
- Landing page with animated introduction
- About Me section with personal information
- Experience section showcasing work history at:
  - Intuit
  - Reazon Holdings
  - Arcules
  - Raytheon
  - iD Tech Camps
- Skills and Technologies showcase
- Software Projects section featuring:
  - Portfolio website
  - Object Detection project
  - Face Morphing
  - Home Server
  - Spotify project
  - Well project
  - Diggerman game
- Art Projects gallery
- Contact modal
- Dark/Light theme toggle with View Transitions API support
- Responsive design for all screen sizes

#### Technical Stack
- React 18 with TypeScript
- Vite build tool
- CSS for styling (no frameworks)
- Strict TypeScript configuration
- ESLint and Prettier for code quality
- Vitest for testing
- Husky for git hooks
- Cloudflare Pages deployment

#### Accessibility
- Keyboard navigation support
- ARIA labels for interactive elements
- Semantic HTML structure
- Focus management

#### Performance
- Optimized asset loading
- WebP image format for photos
- SVG for icons and illustrations
- Font optimization with async loading
