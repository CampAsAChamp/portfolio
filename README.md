## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CampAsAChamp/portfolio.git
   ```
2. Navigate into the repo and run
   ```sh
   npm install
   ```
3. Start the website with
   ```sh
   npm run start
   ```
4. Open your web browser (if not automatically done for you) to `localhost:3000`
